# Shogi KIF Player for Notion

Notionに埋め込んで使える、将棋KIF棋譜ビューアです。  
GitHub上のKIFファイルをURL指定で読み込み、  
ページを開いた瞬間に盤面を表示できます。

## 使い方

1. `kif/` フォルダに `任意のファイル名.kif` ファイルを保存  
2. 以下の形式のURLをNotionに埋め込む：

https://qupffer.github.io/Shogi_kifplayer/?kif=ファイル名.kif
